# Jet ISM

A Python package for analyzing ISM simulations.

## Downloading package

```bash
git clone https://github.com/olgaborodina/jet_ism.git
```

## Installation

```bash
cd jet_ism
pip install .
```

## Usage

```python
from jet_ism import *
from jet_ism.gas import phases

# Your analysis code here
```
